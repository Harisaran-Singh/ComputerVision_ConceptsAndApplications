# ComputerVision_ConceptsAndApplications

This repository contains solutions to a series of Computer Vision and Image Processing assignments implemented using Python and OpenCV.

## Topics Covered

### Assignment 1
- Morphological Transformations
  - Erosion
  - Dilation
  - Opening
  - Closing
- Noise Removal using Morphological Operations
- Sobel Edge Detection
- Canny Edge Detection
- Hough Line Transform
- Fourier Transform
  - Low-Pass Filtering
  - High-Pass Filtering
- Contour Detection

### Assignment 2
- Color Space Analysis
- Histogram Computation and Visualization
- Histogram Equalization
- Contrast Enhancement
- Image Filtering and Smoothing

### Assignment 3
- Image Segmentation
- Thresholding Techniques
- Object Boundary Detection
- Feature Extraction

## Tech Stack

- Python
- OpenCV
- NumPy
- Matplotlib
- Jupyter Notebook
